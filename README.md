# Functions
- Sign-in
- Signup
- Forgot password
- Reset password
- Filter, Search Product
- Add, remove to cart
- Add, remove to favorite
- Add, remove payment method
- Add, remove address shipping
- Checkout
- Notification
- Change information profile
- Add your review
- Dynamic link
